# Projeto Web - Centro Acadêmico de Ciência da Computação

Este projeto foi desenvolvido como parte da cadeira de Desenvolvimento Web no 8º período do curso de Ciência da Computação. O objetivo é criar uma página web funcional e responsiva para o Centro Acadêmico de Ciência da Computação (CA) da universidade, utilizando tecnologias modernas de desenvolvimento web.

## Funcionalidades

- **Página Inicial**: Informações gerais sobre o CA, incluindo sua história, missão e objetivos.
- **Múltiplas Páginas**: Implementação de navegação entre diferentes seções, como eventos, projetos, membros e contato, usando React Router.
- **Menu Hambúrguer**: Menu responsivo que se adapta a diferentes tamanhos de tela.
- **Formulários**: Formulário de cadastro e login de novos membros, com validação de dados.
- **Footer e Header Personalizados**: Seções de header e footer estilizadas, com links para redes sociais e informações de contato.
- **Componentes Reutilizáveis**: Criação de componentes de interface reutilizáveis como botões, ícones e campos de formulário.

## Tecnologias Utilizadas

- **React**: Biblioteca JavaScript para construção de interfaces de usuário.
- **React Router**: Para navegação entre as páginas.
- **CSS**: Para estilização das páginas e ajuste de layout.
- **JavaScript/TypeScript**: Linguagem principal utilizada para a lógica do projeto (ainda em decisão).
- **Figma**: Para design de protótipos e planejamento visual.
